iTerm
===

- 커서 맨 앞으로 이동 <br>
  Control(⌃) + a
- 커서 맨 뒤로 이동 <br>
  Control(⌃)c + e
- 새탭 생성 <br>
  Command(⌘) + t
- 탭 이동 <br>
  Command(⌘) + 방향키
- 탭 닫기 <br>
  Command(⌘) + w
- 현재 커서 앞에 글자들 지우기 <br>
  Control(⌃) + u
- 현재 커서 뒤에 글자들 지우기 <br>
  Control(⌃) + k


# Apple Key Symbols
- Command(⌘)
- Option(⌥)
- Shift(⇧)
- Control(⌃)
- Backspace(⌫)
